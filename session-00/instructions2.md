# Instructions

## Map

[https://travellermap.com](https://travellermap.com)

- Setores (grandes regiões do espaço, com cerca de 32×40 parsecs)
- Subsectors (cada setor é dividido em 16 subsectors)
- Sistemas estelares (cada hexágono do mapa é um sistema)
- Mundos (cada sistema pode ter um ou mais planetas habitáveis)

## Tech Level

| TL    | Descrição                                              |
| ----- | ------------------------------------------------------ |
| 0-2   | Idade da pedra até idade do ferro                      |
| 3-4   | Renascimento, pólvora, navegação                       |
| 5-6   | Industrial, motores a combustão                        |
| 7-8   | Eletrônica avançada, computadores iniciais             |
| 9-10  | Espaço comum, naves orbitais, IA básica                |
| 11-12 | Viagens interestelares maduras, robótica avançada      |
| 13-15 | Megaestruturas, IA forte, engenharia de salto refinada |
